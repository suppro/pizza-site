# Инструкция по созданию диаграмм в Draw.io

## Проблема
Draw.io не поддерживает импорт Mermaid кода напрямую. При импорте SVG из Mermaid получается некрасиво, так как стили не переносятся корректно.

## Решение 1: Импорт SVG с последующей стилизацией

### Шаг 1: Экспорт из Mermaid.live
1. Откройте https://mermaid.live/
2. Вставьте код диаграммы
3. Нажмите "Actions" → "Download SVG"
4. Сохраните файл

### Шаг 2: Импорт в Draw.io
1. Откройте Draw.io (https://app.diagrams.net/)
2. File → Open → выберите SVG файл
3. Диаграмма импортируется, но без стилей

### Шаг 3: Применение стилей Draw.io
1. Выделите все элементы (Ctrl+A)
2. В правой панели найдите раздел "Style"
3. Примените стили:
   - **Фигуры**: Выберите "Rounded" или "Rectangle" с закругленными углами
   - **Цвета**: 
     - Начало процесса: #E1F5FF (светло-голубой)
     - Конец процесса: #C8E6C9 (светло-зеленый)
     - Проблемы/ожидание: #FFCDD2 (светло-красный)
     - Важные операции: #FFF9C4 (светло-желтый)
   - **Шрифт**: Arial, 12pt, жирный для заголовков
   - **Стрелки**: Выберите "Arrow" с закругленными линиями

## Решение 2: Создание диаграммы вручную в Draw.io (рекомендуется)

### Настройка шаблона

1. **Создайте новый файл** в Draw.io
2. **Настройте стили по умолчанию**:
   - View → Style → установите:
     - Font: Arial, 12pt
     - Fill: #FFFFFF
     - Stroke: #000000, 2px
     - Rounded: 1 (для закругленных углов)

### Создание элементов

#### Для процесса приема заказа:

1. **Создайте фигуры**:
   - Используйте "Rectangle" (прямоугольник) для процессов
   - Используйте "Diamond" (ромб) для решений
   - Используйте "Ellipse" (овал) для начала/конца

2. **Примените цвета**:
   - Начало: Ellipse, Fill: #E1F5FF, Stroke: #1976D2
   - Процессы: Rectangle, Fill: #FFFFFF, Stroke: #424242
   - Решения: Diamond, Fill: #FFF9C4, Stroke: #F57F17
   - Конец (успех): Ellipse, Fill: #C8E6C9, Stroke: #388E3C
   - Конец (отмена): Ellipse, Fill: #FFCDD2, Stroke: #D32F2F

3. **Добавьте текст**:
   - Двойной клик на фигуру для редактирования
   - Используйте жирный шрифт для важных элементов

4. **Соедините элементы**:
   - Используйте "Arrow" (стрелка) из панели инструментов
   - Для условий используйте "Yes/No" метки на стрелках

### Готовые стили для копирования

Я создам готовые .drawio файлы, которые можно открыть напрямую в Draw.io. Эти файлы будут содержать уже стилизованные диаграммы.

## Решение 4: Использование PlantUML (альтернатива)

PlantUML лучше интегрируется с Draw.io. Можно использовать плагин PlantUML для Draw.io или экспортировать из PlantUML в Draw.io формат.

## Быстрые советы для красивого оформления

1. **Используйте сетку**: View → Grid → включите сетку для выравнивания
2. **Выравнивание**: Выделите несколько элементов → Arrange → Align
3. **Равномерное распределение**: Arrange → Distribute
4. **Группировка**: Выделите элементы → Arrange → Group (Ctrl+G)
5. **Единый стиль**: Создайте один элемент с нужным стилем, затем копируйте его
6. **Слои**: Используйте слои для организации сложных диаграмм

## Экспорт для дипломной работы

1. **Для Word**: File → Export as → PNG (высокое разрешение, 300 DPI)
2. **Для LaTeX**: File → Export as → PDF или SVG
3. **Для презентации**: File → Export as → PNG (среднее разрешение)

## Рекомендуемый порядок работы

1. Создайте первую фигуру с правильным стилем
2. Скопируйте её для всех похожих элементов
3. Измените только текст
4. Соедините стрелками
5. Выровняйте и распределите элементы
6. Добавьте метки на стрелки (Yes/No)
7. Экспортируйте в нужный формат
