# Быстрый старт: Красивые диаграммы в Draw.io

## Самый простой способ (рекомендуется)

### Вариант 1: Использовать готовый XML файл

1. Откройте файл `drawio_templates/process_1_order_reception.xml` в Draw.io
2. Файл уже содержит стилизованную диаграмму
3. Отредактируйте текст под свои нужды
4. Экспортируйте: File → Export as → PNG (300 DPI)

### Вариант 2: Создать с нуля за 5 минут

#### Шаг 1: Настройте стили (один раз)

1. Откройте Draw.io
2. Создайте первую фигуру (прямоугольник)
3. В правой панели "Style" установите:
   - **Fill**: #FFFFFF (белый)
   - **Stroke**: #424242 (темно-серый), толщина 2px
   - **Rounded**: 10px (закругленные углы)
   - **Font**: Arial, 12pt

4. **Сохраните как стиль**: 
   - Правый клик на фигуру → "Set as Default Style"
   - Или используйте Format Painter (кисточка) для копирования стиля

#### Шаг 2: Создайте библиотеку стилей

Создайте несколько фигур с разными стилями и сохраните их на отдельном листе как шаблоны:

**"Начало/Конец (успех)":**
- Ellipse (овал)
- Fill: #C8E6C9
- Stroke: #388E3C, 2px
- Font: Arial, 14pt, Bold

**"Начало процесса":**
- Ellipse
- Fill: #E1F5FF
- Stroke: #1976D2, 2px
- Font: Arial, 14pt, Bold

**"Процесс":**
- Rectangle (rounded)
- Fill: #FFFFFF
- Stroke: #424242, 2px
- Font: Arial, 12pt

**"Решение":**
- Diamond (ромб)
- Fill: #FFF9C4
- Stroke: #F57F17, 2px
- Font: Arial, 12pt, Bold

**"Проблема/Ожидание":**
- Rectangle (rounded)
- Fill: #FFCDD2
- Stroke: #D32F2F, 2px
- Font: Arial, 12pt

#### Шаг 3: Создайте диаграмму

1. Скопируйте нужные фигуры из библиотеки
2. Измените текст (двойной клик)
3. Соедините стрелками (используйте "Arrow" из панели)
4. Добавьте метки на стрелки (Yes/No) - двойной клик на стрелку

#### Шаг 4: Выровняйте элементы

1. Выделите все элементы (Ctrl+A)
2. Arrange → Align → Center (по центру)
3. Arrange → Distribute → Vertically (равномерно по вертикали)

## Полезные горячие клавиши

- **Ctrl+C / Ctrl+V** - копировать/вставить
- **Ctrl+D** - дублировать элемент
- **Ctrl+G** - сгруппировать элементы
- **Ctrl+Shift+G** - разгруппировать
- **Ctrl+1** - выровнять по левому краю
- **Ctrl+2** - выровнять по центру
- **Ctrl+3** - выровнять по правому краю
- **Delete** - удалить элемент

## Советы для красивого результата

1. **Используйте сетку**: View → Grid → включите
2. **Привязка к сетке**: View → Snap to Grid
3. **Единый размер**: Выделите все прямоугольники → установите одинаковую ширину
4. **Равные отступы**: Используйте "Distribute" для равномерного распределения
5. **Группировка**: Группируйте связанные элементы для удобного перемещения

## Экспорт для дипломной работы

### Для Word:
1. File → Export as → PNG
2. Разрешение: 300 DPI
3. Фон: белый
4. Border: 10px (опционально)

### Для LaTeX:
1. File → Export as → PDF
2. Или SVG (для лучшего качества)

## Если что-то пошло не так

1. **Отменить изменения**: Ctrl+Z
2. **Сбросить стиль**: Format → Reset Style
3. **Очистить все**: Edit → Select All → Delete, затем начните заново

## Альтернатива: Используйте готовые шаблоны

В папке `drawio_templates/` есть готовые XML файлы с уже стилизованными диаграммами. Просто откройте их в Draw.io и отредактируйте текст.
