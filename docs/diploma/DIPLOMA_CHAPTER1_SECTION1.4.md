# 1.4. Выбор технологического стека

На основе проведенного анализа выбран следующий технологический стек для разработки интернет-магазина АО «Арвиай».

## 1.4.1. Серверная технология (Laravel)

Для разработки серверной части выбран фреймворк **Laravel 12** – современный PHP-фреймворк с активным сообществом разработчиков [11; 20].

### Обоснование выбора Laravel

Выбор Laravel обусловлен следующими факторами:

1. **Богатая функциональность из коробки** – встроенная система аутентификации, Eloquent ORM, система миграций, Blade-шаблонизатор, система очередей и уведомлений

2. **Безопасность** – защита от SQL-инъекций, XSS-атак, CSRF-атак, безопасное хранение паролей

3. **Активное сообщество** – большое количество готовых пакетов через Composer, активная поддержка

4. **Соответствие требованиям проекта** – нативная поддержка JSON, простая интеграция с REST API для работы с 1С

5. **Производительность и масштабируемость** – оптимизация для работы с большими объемами данных

## 1.4.2. Система управления базами данных (MySQL)

Выбрана **MySQL 8.0** – реляционная СУБД с открытым исходным кодом [5; 9; 15; 19].

### Обоснование выбора MySQL

- Нативная поддержка JSON для хранения технических характеристик товаров
- Высокая производительность и надежность
- Полная совместимость с Laravel через Eloquent ORM
- Поддержка полнотекстового поиска (FULLTEXT INDEX)
- Широкая поддержка и активное сообщество

Структура базы данных включает таблицы: users, categories, products, product_images, orders, order_items. Все таблицы используют кодировку utf8mb4 для корректной работы с русским языком.

## 1.4.3. Технологии фронтенд-разработки

**Blade** – встроенный шаблонизатор Laravel для создания пользовательского интерфейса. Обеспечивает наследование шаблонов, включение частичных шаблонов, автоматическое экранирование данных.

**Tailwind CSS** – utility-first CSS-фреймворк для стилизации интерфейса. Преимущества: быстрая разработка, гибкость, автоматическое удаление неиспользуемых стилей, встроенная поддержка адаптивного дизайна. Для проекта выбрана цветовая палитра в сине-индиговых тонах [3; 12].

**JavaScript (Vanilla)** – для валидации форм, динамического обновления содержимого и интерактивных элементов интерфейса.

**Vite** – сборщик ресурсов для компиляции и оптимизации CSS и JavaScript файлов.

## 1.4.4. Инструменты разработки

- **Git** – система контроля версий для управления исходным кодом
- **Composer** – менеджер зависимостей PHP для управления библиотеками
- **Guzzle HTTP Client** – библиотека для выполнения HTTP-запросов к REST API 1С
- **PHP 8.2+** – язык программирования с высокой производительностью
